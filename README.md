# Burnaby Hospital Cardiology Clinic Website

https://bh-card-website.web.app/

## Overview

A static informational website for Burnaby Hospital Diagnostic Cardiology department.

This project provides a structured, patient-friendly web presence for a cardiology clinic, including pages for diagnostic testing, consultations, and physician information. The site emphasizes clarity, accessibility, and professional medical presentation.

## Features

- Responsive layout optimized for desktop and mobile
- Clear separation of diagnostic testing and consultation services
- Individual pages for cardiology procedures (ECG, stress testing, Holter monitoring, etc.)
- Consistent styling and reusable layout structure

## Technologies Used

- **HTML5** – semantic page structure
- **CSS3** – layout, responsive design, and styling
- **JavaScript** – client-side interactions (where applicable)
- **Firebase** – website hosting
